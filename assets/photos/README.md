# Team Member Photos

This directory contains profile photos for the team members displayed in the ACP ThreadView.

## Required Photos

The following photos are required (JPG format, recommended size: 64x64px or larger):

- `Ada.jpg` - Ada
- `Eva.jpg` - Eva
- `Harper.jpg` - Harper
- `Hiro.jpg` - Hiro
- `Jess.jpg` - Jess
- `Jim.jpg` - Jim
- `Maya.jpg` - Maya
- `Seal.jpg` - Seal

## Photo Specifications

- **Format**: JPG
- **Recommended Size**: 64x64px minimum, square aspect ratio
- **File Naming**: Capitalized first letter, matching the names above

## Usage

These photos are loaded as embedded assets and displayed in the team members section at the top of the chat panel in the ACP ThreadView. The photos are accessed via the asset system using paths like `photos/Ada.jpg`.
